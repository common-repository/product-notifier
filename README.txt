Stock, Promotions or any kind of alerts/notifications for WooCommerce single product page

== Description ==

Product Notifier is an free plugin that allow user to display message/notification on product single template.You can easily enable/disable notifcation from backend option.
User would be able to set different message for each single product page/template. 
This is a lightweight plugin that you can easily integrate with one click.


= Installation =

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `product-notifier` from Plugins page

== Screenshots ==
